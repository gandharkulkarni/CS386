Homework Description:

This homework assignment focuses on if statements, loops and Processing table data using 2-dimensional arrays in JavaScript.

The sales array represents sales data for months Jan – Dec for the years 2016 – 2018. Each row
represents a year, and each column represents the month.

Part 1:
fCalcRowTotal() : Calculates and returns total sales of the year (row) in 2D array data. This function takes 2 parameters - 2D array and rowIndex.

Part 2:
fCalcColTotal() : Calculates and returns total sales of the month (column) for all years in 2D array data. This function takes 2 parameters - 2D array and colIndex.

Part 3:
fCalcTotals() : Calculates and prints total sales per year and total sales per month in 2D array data. This function takes 1 parameters - 2D array.

Part 4:
fCalcAverageTotals() : Calculates and prints total sales per year, average sales per year, total sales per month, average sales for the month in 2D array data. This function takes 1 parameters - 2D array.
